# Weather App ☁️

A simple and responsive weather application built with **React + Vite + Material UI**.  
It allows users to:

- Search weather by city name
- Get weather using current location 📍
- View temperature, humidity, and weather description

## 🔧 Technologies Used

- React
- Vite
- Material UI
- OpenWeatherMap API
